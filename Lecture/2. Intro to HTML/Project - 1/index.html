Lists:

HTML has two types of lists - ordered and unorderd. Each item of your list has to be enclosed in a tag. The Syntax for creating a list is fairly simple.

Suppose you want to create a list like

The following code will easily do the job for you:

 <li> Item 1 </li>  <li> Item 2 </li>  <li> Item 3 </li> <ul>
